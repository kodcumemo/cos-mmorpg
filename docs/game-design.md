# Monetization Rules

## Core Philosophy

The game must never become pay-to-win.

Players may support the game through advertisements, cosmetic purchases and premium membership, but these systems must never provide combat advantages.

---

## Advertisement Rules

Allowed:

- Support Tokens
- Cosmetic Tokens
- Decoration Tokens
- Character Appearance Unlocks
- House Decorations
- Titles
- Emotes
- Visual Effects

Not Allowed:

- Strength
- Dexterity
- Intelligence
- Vitality
- Wisdom
- Luck
- Damage
- Armor
- Skill Gain
- Skill Levels
- Rare Items
- Legendary Items
- Exclusive Combat Bonuses

---

## Rewarded Advertisements

Advertisement Type:

- Rewarded Ad Only

Not Allowed:

- Banner Ads
- Interstitial Ads

Reason:

MMORPG gameplay must never be interrupted by advertisements.

---

## Premium Membership

Premium Membership may provide:

- Additional Character Slots
- Additional Bank Storage
- Additional House Storage
- Cosmetic Rewards
- Convenience Features

Premium Membership may not provide:

- Damage Bonuses
- Stat Bonuses
- Skill Bonuses
- Combat Advantages

---

## Support Token System

Players may watch advertisements to earn Support Tokens.

Support Tokens may be exchanged for:

- Character Rename
- Cosmetic Items
- House Decorations
- Titles
- Emotes
- Visual Effects

Support Tokens may never be exchanged for power.

---

## Game Balance Rule

A player who never spends money must be able to reach the same combat strength as a paying player.

The difference between players should come from:

- Skill
- Build Choice
- Experience
- Teamwork

Never from payments.

# Server Longevity Rules

- No unlimited power growth.
- Maximum item property count is 5.
- Maximum skill values are capped.
- Endgame progression is horizontal.
- Build diversity is preferred over raw power.
- Event systems must continuously create new goals.
- The strongest player should not reliably defeat three equally skilled players at once.

## Monetization Architecture

Player power and monetization systems are separated.

Combat power is determined by:

- Skills
- Equipment
- Character Stats
- Buffs

Combat power is never determined by:

- Advertisements
- Premium Membership
- Real Money Purchases

Monetization systems use:

- Support Wallet
- Premium Account
- Cosmetic Rewards
- Decoration Rewards

These systems must never affect combat balance.
